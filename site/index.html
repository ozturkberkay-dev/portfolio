Beymen Collection 2025 FW

LoRA Training / Virtual Try-On / Commercial Campaign

Full AI production for the Beymen Collection Fall/Winter 2025–2026 campaign, covering both Women's and Men's collections. All visuals were generated using AI and are currently live on Beymen's website.

Production Workflow

Received original pose references from Beymen's creative team. Replaced human models with custom-trained AI model LoRAs while preserving exact body positioning and garment interaction. Extracted garments from flat-lay product photography and applied them to AI models using virtual try-on techniques, maintaining fabric texture, drape, and color fidelity.

AI Fashion Studio

SaaS Platform / Virtual Try-On / Video Generation

Multi-user web platform for AI-powered fashion visualization. Features a chatbot-style interface where users can upload flat-lay garment images and receive virtual try-on results, enhanced images, and 360° videos all in one workflow.

Platform Capabilities

  • Virtual try-on from flat-lay garment images
  • 1K / 2K / 4K image enhancement pipeline
  • MediaPipe-based face quality improvement
  • 360-degree video generation
  • Clothing catalog with visual search
  • Credit-based usage system with admin panel

Technical Stack

  • Node.js / Express backend with PostgreSQL
  • JWT authentication and session management
  • Vertex AI + Gemini API integration
  • Kling v3 Pro for video generation
  • Railway deployment via GitHub CI/CD
  • Mobile-optimized with sleep-state recovery

Key Screens

Started as a Python/Tkinter desktop pipeline — evolved into a full multi-user SaaS with chatbot UI.

Batch PDP Generation System

Desktop Application / Automation / Product Imagery

Desktop application built for fashion brands to automate product detail page imagery at scale. Select AI models, pair them with garments, hit run - and receive hundreds of studio-quality PDP images in hours instead of weeks.

How It Works

The system chains together multiple AI stages in the background: character sheet generation for consistent AI models, garment fitting from flat-lay product photos, face enhancement, and a library of preset PDP poses. Users only need to select which models and garments to use, set the number of outputs per product (1–5), and start the batch.
